Android群英传1/2章笔记

发表于 2017-10-29 | 分类于 Android

  1. Android四大组件： Activity，BroadCastReciever，ContentProvider，Service

  2. Activity，Service，Application都继承自Content

  3. ADB常用命令:

安装Apk -> adb install -r a.apk 或者 adb push 不同点是install是将Apk安装到data/data目录下，而push命令是将一个文件写入手机存储系统，因此需要相应权限，就可以把Apk放到任何目录下，甚至是System目录下。

向手机写入文件： adb push

从手机获取文件： adb pull

删除应用： adb remount -> adb shell -> cd system/app -> rm a.apk

查看系统盘符： adb shell df

输出所有已经安装的应用程序： adb shell pm list packages -f

模拟按键输入： adb shell input keyevent

模拟欢动输入： adb shell input touchscreen swipe 18 665

启动一个Activity： adb shell am start -n 包名/包名+类名

录制屏幕： adb shell screenrecord /sdcard/a.mp4

重启： adb reboot
